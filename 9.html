Установка

Чтобы начать работать с данной библиотекой, её нужно установить. Для этого нужно прописать в терминал:

pip install pyTelegramBotAPI

ВАЖНО: Нужно качать именно эту библиотеку

Стартовый код

Итак, для начала импортируем основные библиотеки. Получим токен(смотреть ниже). Сделаем команду, которая выводится при старте. Потом сделаем так, Чтобы бот не завершал свою работу.

import telebot
from telebot import types

bot = telebot.TeleBot("2135657858:AAHXgPFoh4qNxZh4MOPTMvNkhUH6BGrdAKQ")

@bot.message_handler(commands=['start', 'Start'])
def write_start(message):
bot.send_message(message.from_user.id, "Текст")

bot.polling()
