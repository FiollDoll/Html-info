Функиции, кнопки и тд.

Начнём с того, как сделать кнопку.
input type ="button", value="Hello world!"

С помощью input мы создаём поле ввода, которому придаём тип "Кнопка.", value - это текст в кнопке. Такая кнопка и получается! Пока на неё можно кликать и ничего не произойдёт. Это надо исправить!!! Изучем теперь создание функций.
script type="text/javascript"
;;;;function showMessage(){
;;;;alert("Иди спи")
;;;;}
script

input type = "button", value ="Кликни меня", onclick="showMessage()"

Мы задаём скрипт и внутри него создаём функцию, давая её название. Потом мы вписываем её содержимое. А теперь, изменения в кнопке. Мы добавляем функцию oncklick(Если нажато) и вписываем туда функцию, готово!!

Математические действия

Время перейти к математике! Опять же показано всё на коде:
script
;;;;function test(){
;;;;;;;;result = 5+5
;;;;;;;;alert(result)
;;;;}
/script

input type="button" value="Калькулятор" onclick="test"
Этот код при нажатии на кнопку выведет число "55". Почему не 10? Потому что язык считывает их за строку. Если же вычитать, то будет 0, всё правильно. Чтобы ответ был 10, нужно заменить строку result = 5+5 на result = Number(5)+Number(5)
