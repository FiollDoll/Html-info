Основы

Итак, покажу основной код, который должен быть обязательно:


"!DOCTYPE html"

"head"
"title"Telebot"/title"
"meta charset = "utf-8""
"/head"

"body"
"/body"
"html"

Итак, !DOCTYPE html - это обозначение что используется версия html. "title"Telebot"/title" - Это название вкладки.

Текст

Чтобы переносить текст на сайт, можно сделать такие правки в коде:
Заголовок пишется таким образом: "h"Заголовок"/h". К h можно прибавлять цифры от 1 до 6. Чем цифра больше, тем меньше.
Можно просто написать текст, после чего он выведится, но лучше писать так: "p" Абзац "/p"
Специальные символы:
"br" - тоже самое что и /n, но в html.
"em""/em" - Курсив текст
"b"Жирный текст"/b"
"li" "/li" - Это типо списка.
Приукрасим li немного. С помощью li type="circle", перед текстом будет круглешок. Нужно обьязательно поставиль "ul".
"ul"
"li type ="circle"
Текст
"/li"
"/ul"

Css

Css - это язык каскадных стилей. Чтобы использовать его, есть много способов. Начну с использования контейнеров:
"!DOCTYPE html"

"head"
"title"Telebot"/title"
"meta charset = "utf-8""

"style"
test{
color: red;
}
"/style"
"/head"

"body"
"test" Красный текст"/test"
"/body"
"html"
 Создаётся контейнер внутри "style" "/style". Мы пишем название контейнера, в этом случае это test. Далее вставляем в него характеристики. Далее в "test" Красный текст"/test" мы используем этот контейнер. Внути "style" "/style" можно создавать хоть сколько контейнеров.
Настройки:
color: ; - задвёт цвет текста
font-size: 100%; - задаёт размер текста
position: relative; left(top, right): 450px; - задаёт позицию по пикселям
text-shadow: 0 0 3px black - создаёт тень

Нужно подсветить текст при наводе на него? Нужно сделать два контейнера, допустим:
style
;;;;test{
;;;;;;;;color:white;
;;;;}
;;;;test:hover{
;;;;;;;;color:black;
;;;;}
/style
Мы создаём обычный контейнер, а потом ещё один. Если указывать hover, то это подсветка при наведении.
